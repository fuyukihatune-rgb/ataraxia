# Ataraxia Works

複数性とともに生きる人のための道具、心を落ち着けるための道具、日々を記録して把握するための道具をつくっています。
このリポジトリは Mosaic・Shift・Still・Anchor・Scale・Ember の6アプリを紹介するランディングページです。

- `index.html` — 1ファイル完結の静的ページ（外部依存なし）
- 紹介アプリ
  - **Mosaic** … 過去の記録・感情のタイムライン → https://mosaic.xdcyw.net
  - **Shift** … 今のフロント管理・申し送り → https://shift.xdcyw.net
  - **Still** … 呼吸ガイドと 5-4-3-2-1 グラウンディング → https://still.xdcyw.net
  - **Anchor** … 制御の二分法で頭の中を整える（ストア派） → https://anchor.xdcyw.net
  - **Scale** … 司法/予備の学習記録・科目バランス → https://scale.xdcyw.net
  - **Ember** … 責めない喫煙記録（本数・きっかけ・在庫） → https://ember.xdcyw.net
- 運営しているプラットフォーム
  - **Empowerment Society X (ESX)** … 自由で開かれた言論プラットフォーム → https://esx.xdcyw.net

## デプロイ（Cloudflare Pages）

1. このリポジトリを Cloudflare Pages に接続
2. ビルド設定：フレームワークなし／ビルドコマンド空欄／出力ディレクトリ `/`
3. カスタムドメイン（例：`xdcyw.net` / `www.xdcyw.net`）を割り当て
